## 🎯 What is BCO?

**Border Collie Optimization** is a nature-inspired metaheuristic algorithm that mimics the brilliant herding behavior of Border Collie dogs! 🐶

Just like these smart dogs gather, stalk, and eye their sheep to guide them home, BCO herds solutions toward the global optimum. It's perfect for solving complex optimization problems when traditional methods struggle.

## ✨ Features

- 🐕 **Smart Herding** - Three dogs (Lead, Left, Right) guide the sheep population
- 👁️ **Eyeing Mechanism** - Prevents getting stuck in local optima
- ⚡ **Fast Convergence** - Finds good solutions quickly
- 🎮 **Easy to Use** - Simple interface, few parameters
- 📊 **19+ Benchmarks** - Built-in test functions included
